# Simulador de Redes

## Instalación

### Requisitos
  - Ubuntu 14.04

### Pasos de Instalación
  - Clonar el repositorio:
  
```bash
$ git clone git@github.com:wincus/network-simulator.git
```
  - Ejecutar el script `setup.sh`

```bash
$ cd network-simulator
$ ./setup.sh
```

  - Iniciar la simulación

```bash
$ make start
```
