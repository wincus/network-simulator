# Simulador de Redes

## Instalación

### Requisitos
  - Ubuntu 14.04

### Pasos de Instalación
  1. Clonar el repositorio:
```bash
$ git clone git@github.com:wincus/network-simulator.git
```
  2. Ejecutar el script `setup.sh`
```bash
$ cd network-simulator
$ ./setup.sh
```
  3. Iniciar la simulación
```bash
$ make start
```
